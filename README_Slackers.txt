Game Title: Slackers

Welcome to Slackers. This README provides the instructions on how to download and run the game, as well as troubleshooting tips and controls.

---

Getting Started:
Since this game is distributed via a .ZIP file format, follow the below steps to get playing:

1. Download the .ZIP to your computer (File Name: SlackersBuild.ZIP)
2, Extract the files: 
	2.1 (Windows) Right-Click on the .ZIP file and select Extract All
	2.2 (macOS) Double-Click on the .ZIP file and the folder will be added above the .ZIP file
	2.3 From here you can move the folder to wherever is convenient for you on your device.
3. Run the game: Open the newly extracted folder, navigate to the .exe file (File Name: Slackers.exe), double-click on the .exe to run the game program.

---

Controls
Below is the basic controls needed to play the game, some controls are explained by prompts in-game:

Player Movement:
Movement: WASD (W - Forward, A - Left, S - Backward, D - Right)

Additional Controls:
Interact With Object: E

Menu Controls:
Interact With Menu Button: Left Mouse Button
Swap Between Menu Buttons: Tab (Non-Essential)

---

Troubleshooting

"The Game Won't Start"
Make sure you have extracted the contents of the .ZIP in its entirety. The .exe needs the accompanying _Data folder and various .dll files in the same directory to function.

"Antivirus Blocked The File"
Due to this being an independent build, antivirus programs such as Windows SmartScreen may flag the .ZIP or .exe files as a virus.
In the event that this happens, proceed to select "More Info" and then "Run Anyway".
Some antivirus softwares have a dedicated 'Exclusion List', so this process may vary depending on the exact software and/or version.
